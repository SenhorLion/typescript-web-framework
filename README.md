# A Typescript Web Framework Project

An Example of building a web framework using Typescript.

### Features:

- Inheritance
- Composition
- Interfaces
- Generics
- Eventing system
- Persisted data
- JSON Server

### Dependencies:

- Node
- Typescript
- Parcel
- JSON-server (for developing)

### Usage:

- Install deps: `npm install`

- Run app with Parcel: `npm run start`
